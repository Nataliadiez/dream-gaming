# 🎮 Dream Gaming — E-commerce Gamer Fullstack

Dream Gaming es una plataforma de e-commerce enfocada en hardware gamer.  
Permite explorar productos, gestionar compras y administrar el catálogo completo desde un panel interno.

No es solo una tienda: es un sistema completo con carrito dinámico, gestión de productos y registro de ventas.

---

## ✨ Vista general

🛒 Catálogo de productos por categoría  
💾 Detalle de producto con información técnica  
🧾 Carrito de compras dinámico  
🎨 Modo claro / oscuro  
🎮 Diseño gamer (pixel + neon vibes)

---

## ⚙️ Instalación

### Clonar repositorio

```bash
git clone https://github.com/Nataliadiez/dream-gaming.git
```
Backend
```bash
cd backend
npm install
npm run start
```
Frontend
```bash
cd frontend
npm install
ng serve
```
